# Project Portfolio Page - Mohamed Faras S/O Faridul Huk

## Overview
**InventoryBRO** is a desktop application designed specifically for small shop owners to manage inventory and daily transactions efficiently using a Command Line Interface (CLI). While the application is optimised for users who prefer typing commands over clicking buttons, it provides a fast, reliable, and precise way to manage stock.
